üìÇ Repository Structure
The repository is organized by assignment modules. Each directory contains the source code for all problems related to that topic.

/Assignment-1 (Array and Matrix Fundamentals):
1.Finding the Minimum Value: Locates the smallest element in an array.

2.Finding the Second Minimum Value: Finds the second unique smallest element.

3.Array Rotation: Implements left/right rotation of array elements.

4.Matrix Operations: Basic operations on 2D arrays (matrices).

5.Finding the Second Largest Value: Locates the second largest element in an array.

Assignment-2 (Sorting and Searching Algorithms):
1.Selection Sort: Implementation of the in-place comparison sorting algorithm.

2.Quick Sort: Implementation of the efficient, divide-and-conquer sorting algorithm.

3.Binary Search: Implementation of the search algorithm for sorted arrays.

4.Linear Search: Implementation of the sequential search algorithm.

5.Bubble Sort: A simple comparison-based sorting algorithm.
